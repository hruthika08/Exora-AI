# 🌌 Exora AI - The Future of Telugu is Voice

Exora AI is a state-of-the-art AI assistant specifically designed for Telugu users. It combines advanced voice recognition, intelligent multimodal analysis, and cultural context to provide a seamless digital experience in the Telugu language.

---

## 🚀 Mission: The Future is Telugu
Telugu is spoken by millions worldwide, yet modern AI often falls short in supporting the language's nuances. Exora AI bridges this "Language Gap" by providing:

*   **Native Telugu Support**: AI that understands and communicates in Telugu naturally.
*   **Empowering Users**: Helping Telugu people use modern technology without language barriers.
*   **Voice-First Interface**: Making interaction as easy as speaking.

---

## ✨ Key Features

### 🎙️ Intelligent Voice Interaction
*   **Voice Input**: Speak naturally, and Exora evaluates your input in Telugu.
*   **Voice Output (TTS)**: For those who prefer listening, Exora reads out its responses in clear Telugu.

### 👁️ Multimodal Vision Capabilities
*   **OCR Support**: Reads Telugu text directly from images.
*   **Image Analysis**: Understands and discusses photos and visual content.
*   **Flexible Uploads**: Select files from your device or use your camera directly.

### 🎨 Creative AI
*   **Image Generation**: Create stunning visual art using integrated AI models.

### 🔒 Security & Privacy
*   **Secure Authentication**: Log in using Google, GitHub, or secure Email/Password verification.
*   **Data Control**: Maintain full control over your chat history with the ability to delete conversation data securely stored in our database.

## 🏗️ Getting Started

### Prerequisites
- Python 3.10+
- Node.js (for frontend development)
- API Keys for Google Gemini and Supabase

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hruthika08/Exora-AI.git
   ```
2. Set up the backend:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
3. Configure your environment variables in `.env`.
4. Run the application:
   ```bash
   python run.py
   ```
